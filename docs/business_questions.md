# Business Questions Solved

## Customer Analysis

- Which customers place above-average orders?
- Which customers have the same ratings?
- Which customers belong to specific cities?
- Which customers generate high-value purchases?

## Salesperson Analysis

- Which salespeople have multiple customers?
- Which salespeople generate the highest order values?
- Which salespeople service customers within their own city?

## Order Analysis

- What is the largest order for each salesperson?
- What is the total value of orders by salesperson?
- Which orders exceed average purchase values?
- Which dates generated the highest order totals?

## Relationship Analysis

- Match customers and salespeople based on location.
- Identify customers assigned to each salesperson.
- Analyze customer-salesperson interactions through orders.
